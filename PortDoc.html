    • Save

    • Export

    • Undo

      You can use ctrl+z for Undo

    • Accidentally close

      If the page is accidentally closed before saving, Reopen the page and press ctrl+z to return to the last state before closing.

    • Eval

      You can use js eval() by select text and on appeared menu click on

      • By double click on evaluated content you can see and change raw data, and by double click on raw data it evals again

      • in table body you can eval and access to table data by using row[i][j] or col[j][i] ex: Number(row[1][1]) + Number(col[2][2])

      • in table header by click on you can select eval and access data by cell value ex: cell.length==2
